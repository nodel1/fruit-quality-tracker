Implementación en Node.js/Python (Docker/PostgreSQL) de una app web y móvil para clasificación de calidad y trazabilidad de frutas mediante visión artificial: backend servidor (Node.js para servicios web y gestión de imágenes/datos, PostgreSQL para almacenamiento de lotes y resultados con consultas SQL para trazabilidad), despliegue portátil vía Docker. Procesamiento AI en Python con OpenCV y scikit-learn para segmentación de imágenes, medición de calibre, detección de forma y color; integración de TensorFlow o PyTorch para entrenamiento y despliegue de modelos de IA. Frontend responsive (HTML/CSS/JavaScript para interfaces operario: carga de fotos, visualización de estadísticas y alertas de defectos). 

## Notas Importantes
- **Modelos de IA Entrenados**: Por respeto a la empresa colaboradora y para proteger los datos sensibles utilizados en el entrenamiento  **los modelos de IA entrenados no se adjuntan en este repositorio**. Estos se generaron durante la estancia en dicha empresa colaboradora y están disponibles solo para uso interno de esta.
- **Instrucciones de Ejecución**: Para iniciar la app correctamente (incluyendo Docker para backend/DB y Cordova para móvil), consulta el archivo **Notes.md** en la raíz del repositorio. Incluye comandos paso a paso para levantar la aplicación.
